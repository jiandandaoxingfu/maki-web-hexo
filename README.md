## 启动

推荐使用`yarn`作为本项目的包管理工具，并在修改了依赖之后也把`yarn.lock`提交上来。

### 安装依赖

```shell
yarn
```

### 启动开发服务器

```
yarn dev
```

## 贡献代码

Commit msg 使用 [conventional-commit](https://www.conventionalcommits.org/en/v1.0.0-beta.2/#why-use-conventional-commits)风格书写，否则会在 lint 阶段报错。
